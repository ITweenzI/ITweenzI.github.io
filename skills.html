Compétences

HTML5

HTML5 est le langage de base pour structurer le contenu web. Il permet de créer des pages web accessibles et interactives. Il est par exemple celui utilisé pour écrire ce site !

CSS3

CSS3 est utilisé pour styliser les pages web, en ajoutant des couleurs, des animations et une mise en page responsive.

JavaScript

JavaScript est un langage de programmation qui permet d'ajouter de l'interactivité aux sites web, comme des animations et des fonctionnalités dynamiques. Il permet de réaliser beaucoup d'éléments agréables pour l'ergonomie du site et donc pour le confort de l'utilisateur mais peut aussi réaliser des traitements complexes.

PHP

PHP est un langage côté serveur utilisé pour le développement web dynamique, souvent avec des bases de données. Il est le pillier des applications notamment dans les entreprises.
Sa maitrise et sa compréhension son éssentiel pour pouvoir développer des outils informatiques plus complexes.

MySQL

MySQL est un système de gestion de bases de données utilisé pour stocker et gérer des données pour des applications web. Simple d'utilisation, il est pourtant essentielles pour tous sites internet pour la gestion des données utilisateurs par exemples.

Pourquoi tout ces langages ?

Il me semblait important de rassembler ces langages que j'ai appris jusqu'ici a utiliser. Il forme les bases de mes connaissances et, pour beaucoup la base de ce que l'on peut faire en développant sur le web. En les maitrisants, ils nous ouvrent la porte du tout est possible nous permettant de répondre a bon nombres de besoins !
